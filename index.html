Dependency Injection

An example is worth a thousand words:

use DI\Annotations\Inject;

class MyClass {
    /**
     * @Inject
     * @var MyService
     */
    private $service;

    public function __construct() {
        \DI\DependencyManager::getInstance()->resolveDependencies($this);
    }

    public function doSomething($id) {
        $object = $service->getSomething($id);
    }
}

Read more at https://github.com/mnapoli/PHP-DI.

Value injection

PHP-DI also supports @Value annotations to inject values:

use DI\Annotations\Value;

class MyClass {
    /**
     * @Value("db.host")
     */
    private $dbHost;

    public function __construct() {
        \DI\DependencyManager::getInstance()->resolveDependencies($this);
    }
}

You can use a .ini file to configure the value to inject:

di.values["db.host"] = "localhost"

Installation

Using Composer?

PHP-DI is published on packagist.org and can be automatically installed with Composer:

{
    "require": {
        "mnapoli/php-di": "dev-master",
    }
}

That's it! You can start using it.

Using Zend Framework?

Check out the Zend Framework quickstart with PHP-DI already configured with examples on zf-quickstart-di.
