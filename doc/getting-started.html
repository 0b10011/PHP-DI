Installation

Install and use with Composer

The easiest way is to install PHP-DI with Composer.

First, create a file named composer.json in your project root:

{
    "require": {
        "mnapoli/php-di": "1.0.*",
    }
}

Then, run the following commands:

$ curl -s http://getcomposer.org/installer | php
$ php composer.phar install

PHP-DI classes will be autoloaded by Composer if you use Composer's autoloading system (which I highly recommend).

Install and use with Zend Framework

Are you using Zend Framework? Check out the official ZF quickstart with Dependency Injection already configured: zf-quickstart-di.

$ git clone git://github.com/mnapoli/zf-quickstart-di.git
